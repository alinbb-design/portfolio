# 李慧琳｜城乡规划作品集

## About Me

武汉华夏理工学院城乡规划专业本科生，具备城乡规划设计、空间数据分析与规划表达能力。熟练运用 ArcGIS、AutoCAD、Photoshop、SketchUp、CityPlan 等专业软件，能够独立完成从前期调研、数据分析到规划设计与成果表达的全过程工作。

在校期间参与乡村规划、居住区规划、城市设计及公共空间设计等多个项目，积累了较强的空间分析与规划实践能力。曾担任项目负责人及组长，负责规划调研组织、GIS空间分析、方案设计及成果汇报等工作，具备良好的团队协作与项目统筹能力。

本人注重以数据支撑规划决策，善于将空间分析、规划设计与视觉表达相结合，探索更科学、更具实施性的规划方案。希望通过持续学习与实践，不断提升专业能力，为城乡高质量发展贡献专业价值。
